Pro Controller SPI Write — Region Probe

Goal: Determine which SPI regions accept writes on this firmware. Per dekuNukem's docs, status byte 0x01 from subcommand 0x11 means "write protected". We'll probe each region with a single-byte write and report the status code.

Each test is self-contained: it reads current value, writes a small change, verifies, then restores the original byte. Total impact on controller behavior: zero.

Tests

Test 1 — Connect

Test 2 — Write to color region (0x6050)

Body color byte. JC Toolkit modifies this region for color customization, so it should be writable on stock firmware.

Test 3 — Re-confirm patch RAM (0x1DB9B)

Should fail with 0x01 "write protected" again, confirming our earlier result was deterministic.

Test 4 — Patch RAM alt slot (0x28100)

DS2 region (where the OTA-updated firmware actually runs from). Probably also write-protected, but worth confirming.
